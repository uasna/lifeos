# LifeOS v31.8 — Rocket Cycle 1 Air Roll Alignment

## Objetivo
Ajustar el ciclo 1 de Rocket League para incluir powershots con ajuste básico de car angle usando air roll, sin convertir air roll en una mecánica nueva.

## Cambios
- Ciclo 1 pasa a: shots simples + powershots + rotación básica 2v2.
- El bloque principal de powershots aclara que air roll se usa solo para cuadrar el carro antes del impacto.
- La aplicación controlada usa el Training Pack `Power shot + Air Roll Shot`.
- Se mantiene la regla: máximo 1 mecánica nueva a la vez.
- Workshop sigue fuera hasta nuevo aviso.
- Ranked / 1v1 / 2v2 sigue opcional.
