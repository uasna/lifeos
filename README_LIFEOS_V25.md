# LifeOS v25 — Rocket League Workshop Normal Maps

Cambios:
- Rocket League ahora trata Workshop como fuente normal de entrenamiento en Epic.
- Solo se programan mapas Workshop normales/safe: dribbling, rings, aerial control, air dribble básico y tiros.
- Se evita programar mapas con modos extra, minijuegos, mutators raros o reglas custom que puedan buguear Epic.
- Las tarjetas Workshop ahora explican cómo usarlas después de una partida sin reiniciar Rocket League.
- Se agregaron badges e instrucciones claras: “Workshop normal”, “Epic safe”, “sin modos extra”.
- Se mantienen Freeplay y 3 partidas 1v1 como fijos; el resto sigue rotando.

Instalación:
1. Extraer encima del proyecto LifeOS.
2. Ejecutar npm run build.
3. Commit y push.
